# 게시판 만들기
Spring Boot + Thymeleaf 게시판 만들기

## 🖥️ 프로젝트 소개
개발자들이 소통할수 있는 가벼운 게시판을 만들었습니다.

### 개발 기간
 - 24.01.08 - 24.01.20

### 🧑‍🤝‍🧑 개발인원
 - 유영무 - 게시판, 게시글, 댓글 CRUD 제작, Thymeleaf 활용한 View 제작

### ⚙️ 개발 환경
 - **Java** : Java 17
 - **Framework** : Spring Boot 3.2.1
 - **DB** : MySQL
 - **ORM** : JPA, JdbcTemplate
